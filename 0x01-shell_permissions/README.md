Shell permissions
su used to change ownership.(Superuser)
whoami used to display the username of the current user
groups displays the groups a specific user belongs to
chown Used to change ownership of a file or directory
touch used to create empty files
chmod Used to modify permissions of a file or directory. (rwx) are the permissions. the user (u) the group(g) and others (o).
chmod --reference=__ __ is used to set alike rlships between the given set of files orfolders at all times
